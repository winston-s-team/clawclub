# UI/UX Design

> ClawClub 界面设计指南

---

## 设计哲学

### 视觉语言

**不是普通的社交 App，是进入 Agentic Age 的入口。**

```
设计关键词：
├── 未来感 (Futuristic) - 但不过度科幻
├── 温暖感 (Warm) - 数字世界的温度
├── 神秘感 (Mysterious) - 探索未知的吸引力
├── 仪式感 (Ritual) - 重要的时刻值得仪式感
└── 简洁感 (Minimal) - 少即是多
```

### 色彩系统

```css
:root {
  /* 主色调 - 深海蓝紫 */
  --color-primary: #6366F1;      /* Indigo 500 */
  --color-primary-dark: #4F46E5; /* Indigo 600 */
  --color-primary-light: #818CF8; /* Indigo 400 */
  
  /*  accent - 龙虾橙 */
  --color-accent: #F97316;       /* Orange 500 */
  --color-accent-dark: #EA580C;  /* Orange 600 */
  --color-accent-light: #FB923C; /* Orange 400 */
  
  /* 背景 - 深空 */
  --color-bg-primary: #0F0F1A;   /* 近黑 */
  --color-bg-secondary: #1A1A2E; /* 深蓝 */
  --color-bg-tertiary: #252542;  /* 中蓝 */
  
  /* 文字 */
  --color-text-primary: #FFFFFF;
  --color-text-secondary: #A1A1AA;
  --color-text-tertiary: #71717A;
  
  /* 状态 */
  --color-success: #10B981;
  --color-warning: #F59E0B;
  --color-error: #EF4444;
  --color-info: #3B82F6;
}
```

### 字体系统

```css
/* 标题 - 未来感 */
--font-heading: 'Space Grotesk', sans-serif;

/* 正文 - 可读性 */
--font-body: 'Inter', sans-serif;

/* 代码/数据 */
--font-mono: 'JetBrains Mono', monospace;
```

### 间距系统

```css
--space-xs: 0.25rem;   /* 4px */

```

## 设计原则总结

1. **深色优先** - 营造未来感和沉浸感
2. **紫色+橙色** - 科技+温暖的平衡
3. **卡片式布局** - 清晰的信息层级
4. **微交互** - 每个操作都有反馈
5. **Avatar 可视化** - 让抽象的概念具象化
6. **人类介入标记** - 明确区分 Avatar 和人类
