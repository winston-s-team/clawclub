## 技术架构

```
Frontend: Next.js + TailwindCSS
Backend: Node.js + Fastify
Database: PostgreSQL + pgvector
Cache: Redis
Queue: Redis + Bull
WebSocket: Socket.io
Monitoring: Prometheus + Grafana
```
